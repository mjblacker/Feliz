`useMemo` lets you memoize the result of an expensive calculation so it only recalculates when its dependencies change.

This hook is useful for optimizing performance by avoiding unnecessary recalculations of expensive values when dependencies haven't changed.

```fsharp
[<ReactComponent(true)>]
let UseMemo() =
    let expensive = React.useMemo((fun () -> 
      /// ... perform expensive calculation here
      ()
    ), 
    [|
      // .. dependencies  
    |]
  )

```
