`useRef` lets you persist values across renders without causing a re-render. It's commonly used to access DOM elements directly.

This hook is useful for accessing DOM nodes, storing mutable values, or keeping track of values that should not trigger a re-render.


```fsharp
[<ReactComponent(true)>]
let UseRef() =
    let ref = React.useRef(None)
    let inputRef = React.useInputRef()
    let eleRef = React.useElementRef()

    // ..
```

:::info

Feliz provides two specialized hooks, `useInputRef` and `useElementRef`, built on top of `useRef` for easier access to input elements and general DOM elements, respectively.

:::

## useInputRef

Can be used for input elements and casts the ref value to `Browser.Types.HTMLInputElement option`.
