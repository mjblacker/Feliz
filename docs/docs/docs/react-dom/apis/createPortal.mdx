`React.createPortal` lets you render children into a DOM node that exists outside the DOM hierarchy of the parent component. This is useful for modals, tooltips, and overlays.

```fsharp

React.createPortal(
  Html.div [
      // ..  component
  ],
  Browser.Dom.document.body // target DOM node
)
```
